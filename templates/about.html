About

  • ●To use this tool: Simply copy and paste the content in the given box and hit the submit button. Each sentence should be entered on a new line.

  • ●If you are checking words in an Excel sheet or similar, you can copy the entire content and paste it here, then hit enter.

  • ●This sentiment tool has a threshold for positive, negative, and neutral sentiments. Positive sentiment is considered when the score is above 0.6, negative sentiment when below 0.4, and neutral sentiment between 0.4 and 0.6. You can adjust these thresholds by contacting the admin of this tool.

  • ●The core concept of this model is logistic regression built on text using NLP techniques. For more information, please contact us.
